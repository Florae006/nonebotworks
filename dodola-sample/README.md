# dodola-sample小机器人

## 使用

本项目删除了源项目下的登录账号信息，主要提供了一些插件的开源和框架配置的示例

## 插件

### getcfrating

获得收到昵称下的cf账号现在的rating

### getcontestinfo

获得**今日**cf/AtCoder/牛客的比赛信息

### gettomorrowcontestinfo

获得**明天**cf/AtCoder/牛客的比赛信息

### gethelpbook

获得使用说明手册

### randomproblem

随机一题（目前只随机一道cf题

### simplereply

简单的回复字典
